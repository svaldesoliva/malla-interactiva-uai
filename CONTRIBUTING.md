# Contribuir en el proyecto

## Código
Puedes contribuir haciendo un fork del proyecto, crear una branch para la función que quieras implementar, hacer push a Github y de ahí pedir un Pull Request.

## Mallas Curriculares
Para contribuir en las mallas, puedes crear la malla en formato `JSON` tu mismo, o responder el [Formulario](https://forms.cloud.microsoft/r/GZ8Pfg4cTD). 
Para los archivos de tipo `colors_CARR.json`, basarse en el esquema de colores de las mallas publicadas en `admision.uai.cl`

Muchas gracias de antemano por tu aporte!
